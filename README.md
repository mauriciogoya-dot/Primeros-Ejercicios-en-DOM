# 🚀 Ejercicios Progresivos de Manipulación del DOM

¡Bienvenido a este repositorio! Aquí vas a encontrar una serie de **10 ejercicios interactivos** desarrollados de forma progresiva. El objetivo de este proyecto fue dominar los fundamentos y las técnicas avanzadas de manipulación del **DOM (Document Object Model)** utilizando **JavaScript Vanilla** (JS nativo), estructurando el contenido con HTML5 y aplicando estilos modernos con CSS3.

---

## 🛠️ Tecnologías y Herramientas Utilizadas

* **HTML5:** Estructuración semántica de cada componente.
* **CSS3:** Diseños modernos (Flexbox, Grid), transiciones suaves y control de visibilidad.
* **JavaScript (ES6+):** Lógica interactiva, escucha de eventos y persistencia de datos.
* **Git & GitHub:** Control de versiones y alojamiento del repositorio.

---

## 📑 Índice de Ejercicios Desarrollados

A continuación, se detallan los desafíos resueltos en orden de complejidad:

### 🔹 Bloque 1: Fundamentos del DOM (Ejercicios 1 - 5)
1. **Cambiar el texto de un título:** Selección básica mediante `getElementById` y uso de `.textContent`.
2. **Cambiar estilos de una caja:** Modificación dinámica de propiedades CSS en línea mediante `.style` y control por clases con `.classList`.
3. **El Contador:** Manejo de variables de estado en memoria (`let`) y sincronización en tiempo real con la pantalla.
4. **Lista de tareas simple:** Creación dinámica de elementos del DOM (`createElement`), inserción de nodos (`appendChild`) y eventos de eliminación selectiva con `.remove()`.
5. **Filtrado de elementos:** Motor de búsqueda en tiempo real usando el evento `input`, bucles `.forEach()` y métodos de filtrado de texto como `.toLowerCase()` e `.includes()`.

### 🔸 Bloque 2: Componentes Interactivos y Avanzados (Ejercicios 6 - 10)
6. **Modal básico:** Control de ventanas emergentes y capas superpuestas gestionando la visibilidad de manera limpia a través de `.classList.add()` y `.classList.remove()`.
7. **Sistema de pestañas (Tabs):** Navegación fluida por secciones utilizando atributos personalizados de HTML5 (`data-*`) leídos dinámicamente mediante `.dataset`.
8. **Acordeón dinámico de FAQ:** Estructura interactiva plegable/desplegable para optimizar el espacio en pantalla utilizando el método de interruptor `.classList.toggle()`.
9. **Validación de formulario en vivo:** Control dinámico y restricción de envíos mediante la escucha de eventos `input` y `submit`, aplicando `event.preventDefault()` y manejo estricto de mensajes de error en vivo para inputs de tipo *text*, *email* y *password*.
10. **Tablero Kanban Profesional:** El proyecto cumbre que integra la API nativa de **Drag and Drop** (`dragstart`, `dragover`, `drop`) para arrastrar tarjetas entre columnas, combinado con persistencia de datos en el navegador mediante **LocalStorage** con serialización de objetos en formato JSON.
