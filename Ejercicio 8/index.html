Preguntas Frecuentes

HTML es el lenguaje de marcado estándar para crear páginas web. Define la estructura de la página y el contenido mediante etiquetas.

CSS es el lenguaje que usamos para diseñar y darle estilo a nuestras páginas web, controlando los colores, fuentes, espaciados y disposiciones.

JavaScript es un lenguaje de programación que permite hacer que las páginas web sean interactivas, añadiendo animaciones, lógica y control dinámico del DOM.

El DOM (Document Object Model) es la estructura en forma de árbol que crea el navegador al cargar el HTML, permitiendo que JavaScript acceda y modifique los elementos de la página.

Un evento es una acción o suceso que ocurre en el navegador, como un clic, el envío de un formulario o la pulsación de una tecla, que JavaScript puede escuchar y responder.
