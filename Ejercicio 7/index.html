Documentación Web

HTML (HyperText Markup Language)

Es el esqueleto de la página web. Define la estructura básica y el código para delimitar el contenido de una página web, como párrafos, títulos, imágenes o tablas.

CSS (Cascading Style Sheets)

Es el traje y el estilo del sitio. Se utiliza para controlar el diseño, los colores, las fuentes y la disposición visual de las estructuras definidas en el HTML.

JavaScript

Es el motor y el cerebro. Permite crear contenido de actualización dinámica, controlar multimedia, animar imágenes y hacer que la interfaz responda a las interacciones del usuario.
